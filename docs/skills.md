# Skills Catalog

This catalog lists the planned Sports Biomech Skills collection.

## Foundation

| Skill | Scope |
| --- | --- |
| `c3d-handling` | Read, inspect, clean, and export C3D motion-capture files. |
| `motion-capture-basics` | Core marker-based motion-capture setup, labeling, filtering, and event handling. |
| `biomech-statistics` | Study design, repeated-measures statistics, effect sizes, and reporting for biomechanics. |
| `spm1d` | Statistical Parametric Mapping workflows for time-normalized biomechanical signals. |
| `biomech-visualization` | Publication-ready plots, joint-angle traces, force curves, and multi-panel biomech figures. |

## Musculoskeletal

| Skill | Scope |
| --- | --- |
| `opensim-core` | OpenSim model setup, scaling, inverse kinematics, inverse dynamics, and analysis tools. |
| `opensim-moco` | Optimal control and predictive simulation workflows with OpenSim Moco. |
| `addbiomechanics` | AddBiomechanics upload, processing, quality review, and downstream export. |
| `opencap-pipeline` | OpenCap capture, markerless processing, and musculoskeletal analysis workflows. |
| `pose2sim` | Pose2Sim calibration, pose ingestion, triangulation, and OpenSim export. |
| `ceinms-workflow` | EMG-informed neuromusculoskeletal modeling with CEINMS. |
| `mujoco-msk` | Musculoskeletal simulation and control workflows using MuJoCo. |
| `shoulder-modeling` | Shoulder complex modeling, scapulothoracic assumptions, and validation checks. |
| `upper-extremity-modeling` | Upper-extremity model selection, scaling, kinematics, and kinetic interpretation. |

## Markerless

| Skill | Scope |
| --- | --- |
| `theia3d-integration` | Theia3D project setup, export handling, and biomechanics pipeline integration. |
| `kinatrax-integration` | KinaTrax data handling and integration with sports biomechanics workflows. |
| `markerless-validation` | Validation study design and error reporting for markerless motion capture. |
| `pose-estimation-pipelines` | General camera calibration, pose estimation, filtering, and quality-control workflows. |

## Sports Specific

| Skill | Scope |
| --- | --- |
| `baseball-pitching` | Pitching biomechanics, kinematic sequence, joint loading, and reporting conventions. |
| `overhead-throwing` | Overhead throwing mechanics across baseball, softball, handball, tennis, and volleyball. |
| `running-gait` | Running gait analysis, spatiotemporal metrics, joint mechanics, and workload interpretation. |
| `jumping-landing` | Jump and landing mechanics, force-plate metrics, asymmetry, and injury-risk screening. |

## Injury Prevention

| Skill | Scope |
| --- | --- |
| `ucl-load-monitoring` | Elbow valgus load estimation, workload tracking, and throwing exposure monitoring. |
| `workload-management` | Acute-chronic workload, session load, and athlete monitoring data workflows. |
| `return-to-play` | Biomechanics-informed return-to-play testing, criteria, and progress tracking. |

## Data Engineering

| Skill | Scope |
| --- | --- |
| `statcast-integration` | Statcast data ingestion, feature engineering, and linkage with biomechanics data. |
| `hawkeye-data` | Hawk-Eye data parsing, synchronization, and analysis workflows. |
| `biomech-database-design` | Database schemas for trials, athletes, sessions, signals, metadata, and derived metrics. |
